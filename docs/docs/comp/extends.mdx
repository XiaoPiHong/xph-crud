# 扩展自定义组件

## 扩展 XphForm 表单项

### 扩展表单项

如果当前 `XphForm` 组件的所有表单项都满足不了你，`xph-crud` 还提供`XphExtendCompPropsProvider`来扩展表单项，你只需要使用`XphExtendCompPropsProvider`来包裹你的应用，并且为其提供一个表单项的映射对象即可，详细请看下方代码块：

<CodeBlock language={"tsx"}>{extendsFormBaseRaw}</CodeBlock>

<pre>

  </DemoWrapper>
</pre>
